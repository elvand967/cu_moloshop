
Запуск сервера:
python manage.py runserver
---

Создание миграций новых моделей:
python manage.py makemigrations

Миграции:
python manage.py migrate
---

список всех зависимостей в виртуальном окружении:
    создать:
pip freeze > requirements.txt
    установить:
pip install -r requirements.txt
---

{"slug": "privacy-policy"}             # Политика конфиденциальности
